# Aspen

An arduino compatible development platform based on the STM32F103 series IC.

## Features

* ARM Cortex M3 @ 72 MHz
* 64 KB/128 KB Flash
* 20 KB RAM
* 32 kHz RTC
* Micro USB
* Arduino compatible headers
* ST-Link header.
